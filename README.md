# Лабораторная работа №1

## По дисциплине: «Скриптовые языки программирования»

### Тема работы

Реализация шифра Цезаря на языке программирования Python с поддержкой английского и русского алфавитов.

---

## 📌 Описание

В рамках лабораторной работы разработана консольная программа, выполняющая шифрование текста методом Цезаря.

Программа:

* Принимает строку открытого текста от пользователя
* Запрашивает величину сдвига (целое число)
* Выполняет посимвольное шифрование
* Поддерживает:

  * Английский алфавит (A–Z, a–z)
  * Русский алфавит (А–Я, а–я)
* Корректно обрабатывает выход за пределы алфавита (циклический сдвиг)
* Символы вне алфавитов (пробелы, цифры, знаки) не изменяются

---

## 🧠 Принцип работы шифра

Шифр Цезаря — это метод подстановки, при котором каждая буква исходного текста сдвигается на фиксированное количество позиций в алфавите.

Формула сдвига:

```
(код_символа − начало_алфавита + сдвиг) mod размер_алфавита + начало_алфавита
```

Это обеспечивает «зацикливание» при выходе за пределы алфавита.

Пример:

| Исходная буква | Сдвиг | Результат |
| -------------- | ----- | --------- |
| A              | +3    | D         |
| Z              | +3    | C         |
| я              | +2    | б         |
